# Orion Rewrite Guide

## Introduction

Orion Rewrite is a JavaScript rewrite script for Mihomo-compatible clients.

Supported clients:

- Clash Mi
- FiClash
- Mihomo
- Clash Meta

---

## Features

- Automatic proxy groups
- AI routing
- Streaming routing
- Telegram routing
- GitHub routing
- Apple routing
- Custom rules
- Easy to maintain

---

## Installation

1. Upload `Orion.js` to GitHub.
2. Copy the Raw URL.
3. Open Clash Mi or FiClash.
4. Add a JavaScript Rewrite.
5. Paste the Raw URL.
6. Update the subscription.
